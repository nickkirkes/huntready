## Scope summary

Build the Colorado overlay-fixture toolchain as the **Group A at-merge deliverable**:
the library extension, the CO builder script, the CO `EXPECTED_CO_RA_ORPHAN_IDS`
constant, pure-function mocked tests, and a closure note + operator runbook. The actual
fixture JSON files (`geometry-overlays.json` + `geometry-overlays-dropped.json`) are
**Group B operator-pending** — they require S05.2 + S05.4 geometries in the production DB
first, then an operator run of the builder. This mirrors S05.2 / S05.4 exactly (CO geometry
DB writes are operator-driven; this story ships code+tests, not generated fixtures).
