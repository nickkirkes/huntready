## Critical adaptation (Phase B) — NOT a verbatim port

Bear's `_split_fused_block_row` (extract_black_bear.py:1370-1453) splits **every** present block cell
on `\n` and **raises** if any cell's part-count ≠ N (the hunt-code count). That is correct for bear,
where a fused row doubled *all* columns. **Big-game fusion is partial-column**: empirical evidence
from the 4 fused rows in the committed artifact shows some cells carry a single *shared* value that
applies to both logical rows. A verbatim port would raise on all 4 big-game fused rows.

The 4 fused rows (block tuple = `(unit, valid_gmus, dates, sex, hunt_code, list)`, hunt_code at idx 4):

| section | hunt_code (2 codes) | unit | valid_gmus | dates (raw) | sex | list |
|---|---|---|---|---|---|---|
| 304 r0 | `D-M-082-O2-R\nD-M-082-O3-R` | None | `82` (1) | `Oct. 24–Nov. 1\nNov. 7–15` (2) | `''` | `A\nA` (2) |
| 304 r1 | `D-M-107-O1-R\nD-F-107-O1-R` | `107\nPlains` (2) | `107` (1) | `Oct. 24–Nov. 3` (1) | `''` | `A\nA` (2) |
| 660 r0 | `A-M-003-O1-M\nA-M-004-O1-M` | `3\n4` (2) | `3, 301\n4, 5` (2) | `Season dates: Sept. 21–29` (1) | `''` | `A\nA` (2) |
| 736 r0 | `A-M-118-O1-R\nA-F-118-O1-R` | `118\nPlains` (2) | `118` (1) | `Oct. 3–11` (1) | `''` | `A\nB` (2) |

**The adapted split rule** (`N` = number of full hunt codes found in the hunt_code cell, ≥ 2):
for each present (`!= _NO_COL`) block cell, split on `\n` → `k` parts:
- `k == N` → **distribute** (part `i` → synthetic row `i`)
- `k == 1` (no `\n`, including empty/`None`) → **broadcast** (same value to all N synthetic rows)
- otherwise (`1 < k < N` or `k > N`) → **fail loud** (`PdfExtractionError`, ADR-001 — genuine misalignment)

Verified: the `{N | 1 | raise}` rule handles all 4 cases correctly (e.g. sec 304 r0 broadcasts
`valid_gmus='82'` to both O2 and O3 codes, distributes the two season windows, distributes `list='A','A'`).
