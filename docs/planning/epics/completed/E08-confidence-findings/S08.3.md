## 🚩 Spec-candidate for the human (flag-and-discuss; PM does NOT edit `architecture.md`)

`architecture.md` §"Response shape" has an internal tension: `sources: SourceCitation[]` (allows empty) vs `meta.data_freshness.{most_recent_source_date, stalest_source_date}` (non-nullable, uncomputable from zero sources). **S08.3 resolved toward non-empty** (`sources.min(1)` in the zod schema + `buildDataFreshness` throws on empty). If a total-coverage-gap response should instead carry **zero** sources, `data_freshness` must become **nullable** in `architecture.md` (and `response.ts` + the zod schema follow). **Decision needed before E09's `get_regulations` resolution logic** — that's the first tool that can produce a zero-source response.
