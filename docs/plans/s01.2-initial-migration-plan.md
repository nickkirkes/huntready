## Tasks

### T1: Create migration file — PostGIS + independent entity tables (~5 min)
**Files:** `supabase/migrations/20260425000000_initial_schema.sql`
**Action:** Create the migration file with PostGIS extension and the 4 independent entity tables that have no FK dependencies on other app tables
**Details:**

Create the file with these sections in order:

1. `CREATE EXTENSION IF NOT EXISTS postgis;` — MUST be first statement
2. `regulation_record` table — composite PK `(state, jurisdiction_code, species_group, license_year)`:
   - `state text NOT NULL`
   - `jurisdiction_code text NOT NULL`
   - `species_group text NOT NULL`
   - `license_year integer NOT NULL`
   - `schema_version integer NOT NULL DEFAULT 2`
   - `source jsonb NOT NULL`
   - `ingested_at timestamptz NOT NULL DEFAULT now()`
   - `confidence text NOT NULL CHECK (confidence IN ('high','medium','low'))`
   - `additional_rules jsonb NOT NULL DEFAULT '[]'`
   - `PRIMARY KEY (state, jurisdiction_code, species_group, license_year)`
3. `season_definition` table — text PK `id`:
   - `id text PRIMARY KEY`
   - `name text NOT NULL`
   - `opens date NOT NULL`
   - `closes date NOT NULL`
   - `weapon_type text CHECK (weapon_type IN ('any_legal_weapon','archery','rifle','muzzleloader','shotgun','handgun','crossbow','traditional_handgun','heritage_muzzleloader'))`
   - `residency text CHECK (residency IN ('resident','nonresident','both'))`
   - `closure_predicate jsonb`
   - `verbatim_rule text NOT NULL`
   - `page_reference text`
   - `source jsonb NOT NULL`
4. `license_tag` table — text PK `id`:
   - `id text PRIMARY KEY`
   - `license_code text NOT NULL`
   - `name text NOT NULL`
   - `kind text NOT NULL CHECK (kind IN ('general','limited_draw','over_the_counter','statewide'))`
   - `species text NOT NULL`
   - `weapon_types text[] NOT NULL`
   - `residency text NOT NULL CHECK (residency IN ('resident','nonresident','both'))`
   - `quota integer`
   - `quota_range int4range`
   - `purchase_url text NOT NULL`
   - `draw_spec_key jsonb` — soft FK, nullable
   - `reserved_pools jsonb NOT NULL DEFAULT '[]'`
   - `verbatim_rule text NOT NULL`
   - `source jsonb NOT NULL`
5. `draw_spec` table — composite PK `(state, hunt_code, year)`:
   - `state text NOT NULL`
   - `hunt_code text NOT NULL`
   - `year integer NOT NULL`
   - `schema_version integer NOT NULL DEFAULT 2`
   - `quota integer`
   - `point_system jsonb`
   - `residency_cap jsonb`
   - `choices jsonb NOT NULL`
   - `pools jsonb NOT NULL` — NO DEFAULT (empty array is semantically invalid)
   - `draw_phase text NOT NULL CHECK (draw_phase IN ('primary','secondary','leftover'))`
   - `successor_hunt_code_key jsonb` — soft self-FK, nullable
   - `application_deadline date NOT NULL`
   - `parameters jsonb` — escape hatch; shared code NEVER reads
   - `source jsonb NOT NULL`
   - `PRIMARY KEY (state, hunt_code, year)`

**Verify:** `head -80 supabase/migrations/20260425000000_initial_schema.sql` shows PostGIS extension + 4 CREATE TABLE statements with correct columns and constraints

### T2: Add remaining entity tables — reporting_obligation + geometry (~3 min)
**Files:** `supabase/migrations/20260425000000_initial_schema.sql`
**Depends on:** T1
**Action:** Append `reporting_obligation` and `geometry` tables to the migration file
**Details:**

Append these tables after the T1 tables:

1. `reporting_obligation` table — text PK `id`:
   - `id text PRIMARY KEY`
   - `kind text NOT NULL CHECK (kind IN ('harvest_report','mandatory_check','tooth_submission','hide_skull_presentation','cwd_sample','other'))`
   - `deadline text NOT NULL`
   - `deadline_hours integer`
   - `submission_method text NOT NULL CHECK (submission_method IN ('online','phone','in_person_check_station','mail','agency_office'))`
   - `submission_url text`
   - `submission_phone text`
   - `applies_to_regions text[]`
   - `what_to_present text[]`
   - `verbatim_rule text NOT NULL`
   - `source jsonb NOT NULL`
2. `geometry` table — text PK `id`:
   - `id text PRIMARY KEY`
   - `name text NOT NULL`
   - `kind text NOT NULL CHECK (kind IN ('hunting_district','gmu','portion','bmu','cwd_zone','restricted_area','bma','other'))`
   - `geom geography(MultiPolygon, 4326) NOT NULL` — MUST be `geography` type, MUST be `MultiPolygon`, NOT `Polygon`, NOT `geometry`
   - `state text NOT NULL`
   - `license_year integer` — nullable (null = year-invariant geometries)
   - `source jsonb NOT NULL`

### T4: Add link tables (~3 min)
**Files:** `supabase/migrations/20260425000000_initial_schema.sql`
**Depends on:** T3 (all referenced parent tables must exist)
**Action:** Append the 3 link tables that bridge `regulation_record` to child entities
**Details:**

Append these 3 tables:

1. `regulation_season` — composite PK of all 5 columns:
   - `state text NOT NULL`
   - `jurisdiction_code text NOT NULL`
   - `species_group text NOT NULL`
   - `license_year integer NOT NULL`
   - `season_definition_id text NOT NULL REFERENCES season_definition(id)`

   - Same 4 regulation_record columns
   - `license_tag_id text NOT NULL REFERENCES license_tag(id)`
   - Composite PK + composite FK same as above

3. `regulation_reporting` — same shape:
   - Same 4 regulation_record columns
   - `reporting_obligation_id text NOT NULL REFERENCES reporting_obligation(id)`
   - Composite PK + composite FK same as above

CREATE INDEX jurisdiction_binding_geometry_id_idx ON jurisdiction_binding (geometry_id);
